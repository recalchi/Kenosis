# Release Checklist - Kenosis

## 1. Status

Release ainda não é foco da Sprint 0.1. Este checklist existe para preparar o caminho quando houver uma build jogável.

## 2. Protótipo Técnico

- [ ] Versão interna definida.
- [ ] Cena principal definida.
- [ ] Projeto abre no Godot 4.6.
- [ ] Build Windows exportada, se export for habilitado.
- [ ] Build abre fora do editor.
- [ ] Sem crash crítico conhecido.
- [ ] Checklist de QA da Sprint 0.1 concluído.

## 3. Vertical Slice

- [ ] Duração próxima de 5 minutos.
- [ ] Puzzle completo.
- [ ] Leitura de cicatriz.
- [ ] Momento de stealth/fuga.
- [ ] Áudio temporário ou semi-final.
- [ ] Arte inicial consistente.
- [ ] Testado em mais de uma máquina, se possível.

## 4. Loja/Site Futuro

- [ ] Descrição curta.
- [ ] Descrição longa.
- [ ] Screenshots.
- [ ] Trailer.
- [ ] Ícone.
- [ ] Key art.
- [ ] Política de privacidade, se aplicável.
- [ ] Termos de uso, se aplicável.

## 5. Marketing Futuro

- [ ] Anúncio pronto.
- [ ] Posts prontos.
- [ ] Press kit pronto.
- [ ] Página oficial atualizada.
