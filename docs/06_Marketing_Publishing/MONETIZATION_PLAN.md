# Monetization Plan - Kenosis

## 1. Status

Monetização congelada até depois do vertical slice.

## 2. Motivo

Kenosis tem proposta narrativa, melancólica e imersiva. Decisões de monetização tomadas cedo demais podem distorcer gameplay, UX e ritmo de produção.

## 3. Modelos Possíveis Para Reavaliar Depois

| Modelo | Vantagem | Risco |
|---|---|---|
| Premium barato | Simples e coerente com indie narrativo | Menor alcance inicial |
| Demo grátis + jogo pago | Bom para Steam | Exige demo forte |
| Free-to-play com cosméticos | Baixa barreira de entrada | Precisa volume e pode conflitar com tom |
| DLC/expansão | Preserva experiência base | Só funciona se jogo base performar |

## 4. Decisões Pendentes

- Terá loja interna?
- Terá skins?
- Terá anúncios no mobile?
- Terá versão premium sem anúncios?
- Como monetizar sem prejudicar narrativa?
