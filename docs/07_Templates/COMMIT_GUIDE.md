# Guia de Commits — Kenosis

## Formato

```txt
tipo(área): descrição curta
```

## Tipos

- feat: nova funcionalidade
- fix: correção
- docs: documentação
- art: assets visuais
- audio: áudio/música
- refactor: refatoração
- test: testes
- build: build/export
- chore: organização

## Exemplos

```txt
feat(player): adiciona movimento lateral básico
fix(camera): corrige tremor ao seguir player
docs(gdd): define loop principal do vertical slice
art(environment): adiciona teste visual da metrópole ancestral
```
