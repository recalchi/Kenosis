# Kenosis

Projeto: **Kenosis / Exorigem: O Prequel 2D**
Engine: **Godot 4.6**
Repositório: `https://github.com/recalchi/Kenosis`
Local: `E:\Projetos\Godot\Kenosis`

Kenosis é uma aventura narrativa 2D side-scrolling com metroidvania lite, ambientada como prequel do universo Exorigem. O projeto começa PC-first, offline, com protótipo técnico focado em movimento, interação, Ressonância por cooldown e uma sala neutra de teste.

## Estrutura

- `game/`: projeto Godot.
- `docs/`: documentação de produção, design, técnica e publicação.
- `tools/`: scripts auxiliares futuros.
- `builds/`: builds locais e artefatos exportados.
- `marketing/`: materiais de divulgação futuros.
